# Portfolio - Carlos Piedras

Portfolio personal desarrollado con React y Bootstrap 5.

## Características

- Responsive y adaptable a dispositivos móviles
- Soporte multi-idioma (Español/Inglés)
- Temas claro y oscuro
- Componentes para mostrar experiencia, educación, habilidades y proyectos
- Construido con **Vite**
- Emails con **EmailJS**

## Instalación

1. Instalar dependencias:
```
npm install
```

2. Ejecutar en modo desarrollo:
```
npm run dev
```

3. Construir para producción:
```
npm run build
```

## Tecnologías

- **React** - Framework de JavaScript
- **Bootstrap 5** - Framework CSS
- **Smooth Scrollbar** - Scrollbar personalizado
- **Swiper** - Librería para sliders
- **EmailJS** - Servicio de emails
- **Font Awesome** - Iconos
- **PrimeIcons** - Iconos adicionales
